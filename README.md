# Expencify

**Expencify** is an offline-first, intelligent expenditure tracker and analyzer that automates expense management natively using Secure Local AI inference pipelines.

## 🚀 Key Features

*   🤖 **Local AI Categorization**: Fast, secure, on-device expenses sorting using absolute private Local Large Language Models (LLMs like Gemma/Qwen). No user data ever leaves the device workspace buffer!
*   📩 **Instant SMS parsing**: Real-time broadcast hooks detect incoming bank notifications and register them instantly to minimize manual insertion overhead leaks.
*   🔒 **App Vault Security**: Encrypts and shields local dashboard widgets with Biometric / PIN locks framework gates implicitly.
*   📊 **Insights & Reports**: Seamlessly summarizes layout aggregates by Category buckets without duplicate nodes discrepancies.
*   🔋 **Background Service Reliability**: Smooth long-polling routines prevent Android OS setup from killing background message tasks.

---

## 🛠️ Requirements & Setup

To guarantee 100% accurate background SMS detection loops, the application demands:
1. **SMS Receiving Permission** (`Permission.sms`)
2. **Ignore Battery Optimizations** (`Permission.ignoreBatteryOptimizations` trigger to bypass power saver rules).
